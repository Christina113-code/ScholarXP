# ScholarXP — Design System

A sleek, playful math learning interface inspired by Duolingo, but with a calmer, more refined aesthetic.

---

## ✨ Design Principles

### 1. Playful Minimalism

- Keep interactions fun but avoid overwhelming color usage
- Use subtle motion and microinteractions instead of bright visuals

### 2. Soft Geometry

- Rounded corners everywhere (buttons, cards, inputs)
- Friendly shapes that feel approachable

### 3. Focus First

- Content (math problems) is the center of attention
- UI fades into the background when not needed

### 4. Reward Through Motion

- Use animations, not loud colors, to celebrate progress

---

## 🎨 Color Palette

### Primary

- Slate Blue: `#5C6AC4`
- Soft Indigo: `#7F8CFF`

### Neutral Base

- Background: `#F7F8FA`
- Card: `#FFFFFF`
- Border: `#E6E8EC`

### Text

- Primary: `#1F2937`
- Secondary: `#6B7280`

### Accent (Used Sparingly)

- Success: `#22C55E`
- Error: `#EF4444`
- Highlight: `#E0E7FF`

---

## 🔤 Typography

- Font Family: Inter / SF Pro / system-ui

### Scale

- Heading: 24–32px (Semi-bold)
- Subheading: 18–20px
- Body: 14–16px
- Micro: 12px

---

## 🧱 Layout System

### Grid

- 8px spacing system
- Max width: 420px (mobile-first feel)

### Card Layout

- Padding: 16–20px
- Border radius: 16–20px
- Shadow: subtle (0 4px 12px rgba(0,0,0,0.05))

---

## 🔘 Components

### Buttons

- Rounded: 999px (pill shape)
- Height: 44–52px
- Primary: filled (slate blue)
- Secondary: outlined

States:

- Hover: slight lift + shadow
- Active: scale down (0.97)

---

### Progress Bar

- Height: 8px
- Rounded ends
- Soft gradient fill

---

### Answer Choices

- Card-style buttons
- Large tap targets
- Slight hover/press animation

---

### Input Field

- Rounded corners (12px)
- Soft border
- Focus glow (light indigo)

---

## 🎮 Interaction Design

### Microinteractions

- Button press: quick scale animation
- Correct answer: gentle bounce + green check
- Wrong answer: slight shake + red highlight

### Transitions

- Duration: 150–250ms
- Easing: ease-out

---

## 🧠 Gamification Elements

- XP points (minimal visual clutter)
- Streak indicator (small flame icon)
- Daily goals (simple progress ring)

---

## 📱 Screens

### 1. Home Screen

- Progress overview
- Daily goal
- Resume lesson button

### 2. Lesson Screen

- Math question centered
- Answer choices below
- Progress bar on top

### 3. Results Screen

- Score
- XP gained
- Continue button

---

## 🖼️ Visual Style Notes

- Avoid heavy gradients
- Prefer soft shadows over bold outlines
- Keep whitespace generous
- Use icons sparingly

---

## ⚙️ Future Enhancements

MathLingo blends:
